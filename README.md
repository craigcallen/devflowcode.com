# devflowcode.com

Landing page for [DevFlow](https://devflow.app) by Allen Media.

## Stack

- Vite + React + TypeScript + Tailwind CSS v4
- Framer Motion for scroll animations
- Resend for email waitlist capture
- AWS Amplify for hosting
- AWS Lambda for waitlist API

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
